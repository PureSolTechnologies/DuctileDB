Use DuctileDB from Maven

Attention: DuctileDB is not released, yet. So there is not artifact to be found in Maven Central. As soon as the first release is out, the artifacts are put into Maven Central.

The maven artifacts to be used within simple Java SE application are

com.puresoltechnologies.ductiledb ductiledb-core ${ductiledb.version} ]]>

if DuctileDB graph is used without any processing framework. To use Tinkerpop (and DuctileDB graph directly), use dependency

com.puresoltechnologies.ductiledb ductiledb-tinkerpop ${ductiledb.version} ]]> and if also the integration with eXtended Objects is needed, choose: com.puresoltechnologies.ductiledb ductiledb-xo ${ductiledb.version} ]]>
